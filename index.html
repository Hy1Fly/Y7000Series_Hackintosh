Y7000Series Hackintosh

📖 项目概述

为联想拯救者Y7000系列笔记本（2018-2020款）提供经过验证的Hackintosh EFI配置，支持macOS 10.13至最新版本的系统OTA升级。

🖥️ 兼容机型

2018款

  • Y7000
  • Y7000P
  • Y9000K

⚙️ BIOS设置

  1. 启用 UEFI Boot
  2. 禁用 Secure Boot
  3. 设置SATA模式为 AHCI
  4. 执行 BIOS高级设置脚本

✅ 功能支持

完美工作

  • ⌨️ 键盘支持
  • 💤 睡眠唤醒

有限支持

  • 🖥️ HDMI输出
  • 🔌 Type-C扩展
